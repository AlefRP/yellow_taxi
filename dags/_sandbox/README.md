# Sandbox DAGs

DAGs de estudo e demonstração de features do Airflow (XCom, Pools, Sensors,
Datasets, etc.).

**Não são executadas pelo scheduler** — esta pasta é ignorada via `.airflowignore`
no diretório raiz de DAGs.

Para executar uma destas DAGs localmente, mova o arquivo para `dags/`
temporariamente.
